# 什麼是 Markdown？

Markdown 是一種在網頁上樣式化文本的方法。您可以控制文檔的顯示；例如將文字設為粗體或斜體、添加圖片以及創建清單，這些只是我們可以用 Markdown 做的一部分事情。大多數情況下，Markdown 只是普通文本，並添加了一些非字母字符，例如 `#` 或 `*`。

# 語法指南

以下是您可以在 GitHub.com 或自己的文本文件中使用的 Markdown 語法概覽。

## 標題

```
# 這是一個 h1 標籤

## 這是一個 h2 標籤

#### 這是一個 h4 標籤
```

# 這是一個 h1 標籤

## 這是一個 h2 標籤

#### 這是一個 h4 標籤

## 強調

```
_這段文字將是斜體_

**這段文字將是粗體**

_您 **可以** 結合使用它們_
```

_這段文字將是斜體_

**這段文字將是粗體**

_您 **可以** 結合使用它們_

## 清單

### 無序清單

```
- 項目 1
- 項目 2
  - 項目 2a
  - 項目 2b
```

- 項目 1
- 項目 2
  - 項目 2a
  - 項目 2b

### 有序清單

```
1. 項目 1
1. 項目 2
1. 項目 3
   1. 項目 3a
   1. 項目 3b
```

1. 項目 1
1. 項目 2
1. 項目 3
   1. 項目 3a
   1. 項目 3b

## 表格

您可以通過組合單詞列表並用連字符 `-`（第一行）分隔它們，然後用管道符號 `|` 分隔每一列來創建表格：

```
| 第一個標題                | 第二個標題                |
| ------------------------- | ------------------------- |
| 單元格 1 的內容           | 單元格 2 的內容           |
| 第一列中的內容            | 第二列中的內容            |
```

| 第一個標題                | 第二個標題                |
| ------------------------- | ------------------------- |
| 單元格 1 的內容           | 單元格 2 的內容           |
| 第一列中的內容            | 第二列中的內容            |

## 刪除線

任何用兩個波浪號包裹的單詞（例如 `~~這個~~`）將顯示為 ~~刪除線~~。
